# 清單生活 AI - AI 生活行政助手 MVP

這是一個可立即展示的靜態 SaaS 前端 MVP，鎖定台灣個人用戶，主打 AI 文件整理、帳單/繳費/行政提醒、表單草稿與不留原檔資料策略。

## 使用方式

直接以瀏覽器開啟 `index.html` 即可使用，不需要安裝套件或啟動後端。

工作台支援三種範例：

- 電信帳單
- 保單通知
- 政府補助公告

也可以自行貼上帳單、保險、稅務、信用卡、訂閱或行政通知文字，按下「AI 整理文件」產生 demo 解析結果。

## MVP 範圍

- 文件類型分類：報稅、保險、信用卡、訂閱、帳單、政府補助、行程提醒、其他文件
- 擷取欄位：機構、金額、期限/日期
- 輸出內容：待辦卡片、摘要、表單草稿、追溯依據
- 資料策略：Demo 不保存原始檔，只展示結構化摘要與提醒

## 後續接 API 建議

`app.js` 的 `analyzeDocument()` 目前是本機規則模擬。正式版可將此函式替換為後端 API：

1. 前端送出使用者貼上的文字或 OCR 後文字。
2. 後端呼叫 AI 模型回傳固定 JSON schema。
3. 後端丟棄原始檔，只保存使用者確認後的結構化欄位。
4. 前端沿用目前的結果渲染函式。
